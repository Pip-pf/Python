Python Phonebook - Portfolio Project
This is a small Python program used to manage contacts using dictionaries. Users can add, view, update and delete contacts.
Built to practice Python concepts like loops, conditionals, inputs and data structures. inspired by freeCodeCamp's python course.

v1 - Basic phonebook using While loop and if statements

v2 - Using functions instead of if statements and implementing a dictionary of functions, input validation and view all contacts feature
